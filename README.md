# PRODIGY_WD_01 - Responsive Landing Page

## 📌 Project Overview
This repository contains my submission for **Task 01** of the Web Development Internship at [Prodigy InfoTech]. It is a fully responsive landing page designed to adapt seamlessly across different screen sizes and devices.

## 🚀 Features
* **Responsive Design:** Adapts fluidly to mobile, tablet, and desktop viewports.
* **Clean UI/UX:** A structured and visually appealing layout.
* **Interactive Elements:** Dynamic behavior handled via JavaScript.

## 🛠️ Technologies Used
* **HTML5:** Structured the content of the landing page.
* **CSS3:** Styled the layout and implemented media queries for responsiveness.
* **JavaScript:** Added basic interactivity and dynamic features.

## 📂 File Structure
* `index.html` - The main HTML file containing the structural layout.
* `style.css` - The CSS file containing all the styling and media queries.
* `script.js` - The JavaScript file for interactive elements.
